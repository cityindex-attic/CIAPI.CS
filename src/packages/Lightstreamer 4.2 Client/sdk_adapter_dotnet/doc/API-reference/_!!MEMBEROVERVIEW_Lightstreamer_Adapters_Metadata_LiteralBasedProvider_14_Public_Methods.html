Returns the maximum allowed length for a Snapshot of any Item that has been requested with publishing Mode DISTINCT.  
 
GetItems 
Resolves a Group name supplied in a Request. The names of the Items in the Group are returned. Group names are expected to be formed by simply concatenating the names of the contained Items, in a space separated way.  
 
GetSchema 
Resolves a Schema name supplied in a Request. The names of the Fields in the Schema are returned. Schema names are expected to be formed by simply concatenating the names of the contained Fields, in a space separated way.

Reads configuration settings for user and resource constraints. If some setting is missing, the corresponding constraint is not set.
