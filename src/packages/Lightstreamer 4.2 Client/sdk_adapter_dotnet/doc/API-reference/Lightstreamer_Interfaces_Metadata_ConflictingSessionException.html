ConflictingSessionException Class
ConflictingSessionException Properties  ConflictingSessionException Members  Classes

Thrown by the NotifyNewSession method of IMetadataProvider if a User is not enabled to open a new Session but he would be enabled as soon as another Session were closed. By using this exception, the name of the other Session is also supplied. 

After receiving this exception, the Server may try to close the specified session and invoke NotifyNewSession again.
